[发布页面](https://github.com/NateScarlet/auto-dragalia/releases)

## 功能

- [x] 重复单人战斗

  - [x] 自动打开自动战斗开关

  - [x] 自动龙化

    - [x] 只在周围有敌人的时候龙化

  - [x] 自动使用龙技能

  - [x] 自动使用支援技能

- [x] 自动喂龙

- [x] 刷稀有物品(金币任务刷幸运草)

  - [x] 载入第二张图后如果没刷到稀有物品直接放弃任务

- [ ] 重复多人战斗
